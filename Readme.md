## Peanut
学习英语单词就像吃花生米一样，粒粒皆辛苦

## Features
1. 作为数据记录中枢，列出最近看到的生词
2. 按照一定的规则显示生词，如想不起来则可查看详细解释
3. 已经掌握的单词入到单词库中
4. 再次被添加到生词本的单词会有加权处理
5. 可以替换主题

## 输入端：
1. Mac
    - 浏览器内选中生词，配合Alfred的workflow上传到生词服务
    - 在Alfred的快捷输入里拼写单词，然后上传到生词服务
2. iPhone/iPad
    - 浏览器内选中生词，配合使用捷径上传到的生词服务

## 服务端
### 可选方案
1. 使用iCloud服务
2. 用Swift搭一个Server-Side服务
